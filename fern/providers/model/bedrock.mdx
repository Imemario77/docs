title: AWS Bedrock
subtitle: Use AWS Bedrock hosted models with Vapi
---

AWS Bedrock provides access to foundation models through a fully managed service. You can use your AWS Bedrock credentials to access Anthropic's Claude models through Vapi.

## Configuration

1. Go to the [Dashboard](https://dashboard.vapi.ai) under the **Provider Keys** tab
2. Add your AWS credentials:
   - AWS Access Key ID
   - AWS Secret Access Key
   - AWS Region (where Bedrock is enabled)

## Available Models
- anthropic.claude-3-5-sonnet-20240620-v1:0

## Usage

To use AWS Bedrock with Vapi, you'll need to:
1. Have AWS Bedrock enabled in your AWS account
2. Have access to Anthropic models in your AWS Bedrock configuration
3. Configure your AWS credentials in the Vapi dashboard

Example configuration:
```json
{
  "model": {
    "provider": "bedrock",
    "model": "anthropic.claude-3-5-sonnet-20240620-v1:0"
  }
}
```
